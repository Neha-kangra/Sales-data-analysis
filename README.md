# Sales-data-analysis
Interactive Power BI dashboard for retail sales analysis.

Sales Data Analysis Dashboard

Overview

Built an interactive 5-page Power BI dashboard to analyze retail sales data. The project includes Python-based data cleaning, KPI tracking, and interactive visualizations for sales, profit, product, and regional performance.

Tools

Power BI
Python (Pandas, NumPy)

Features

📈 Executive Dashboard
💰 Sales Analysis
📊 Profit Analysis
📦 Product Analysis
🌍 Regional Analysis
Interactive Filters & Page Navigation
